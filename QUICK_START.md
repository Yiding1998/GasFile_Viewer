# 快速使用 / Quick Start

[中文](#中文) | [English](#english)

## 中文

**无需安装软件，无需下载项目，也无需配置运行环境。**

### 直接查看或绘制气体文件

[点击打开中文 Garfield 气体文件工作台](https://yiding1998.github.io/GasFile_Viewer/garfield_gas_workbench_pro.html)

打开后，将一个或多个 `.gas` 文件拖入网页，即可查看、比较和绘图。

### 直接查找气体文件

[点击打开气体文件检索程序](https://yiding1998.github.io/GasFile_Viewer/gas_file_search.html)

输入气体成分、比例、温度或压强；找到合适文件后，点击 **中文工作台**，文件会直接加入工作台。

> 请使用 Chrome、Edge 或 Firefox 等现代浏览器。请打开上面的在线程序链接，不要使用 GitHub 的源代码预览页面。

## English

**No installation, repository download, or environment setup is required.**

### Open or Plot Gas Files

[Open the English Garfield Gas Workbench](https://yiding1998.github.io/GasFile_Viewer/garfield_gas_workbench_pro_english.html)

Drag one or more `.gas` files into the page to inspect, compare, and plot them.

### Find a Gas File

[Open Gas File Search](https://yiding1998.github.io/GasFile_Viewer/gas_file_search.html)

Enter gas components, fractions, temperature, or pressure. Select **English workbench** on a suitable result to add it directly to the workbench.

> Use a modern browser such as Chrome, Edge, or Firefox. Open the online application links above, not GitHub's source-code preview pages.
