# GasFile_Viewer

The gas files are generated by Garfield++, and viewing gas files through a web-based program ([garfield_gas_workbench_pro.html](https://github.com/Yiding1998/GasFile_Viewer/blob/main/garfield_gas_workbench_pro.html "garfield_gas_workbench_pro.html"), `Made with the help of GPT5.5`)

Download the webpage program file and open it directly in browser.
